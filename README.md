# all-women-social-media-platform
A social media platform built exclusively for women to connect, share, support, and grow in a safe and empowering digital space all while focusing on security, privacy, and meaningful community interaction.

## Key Goals
- Create a **secure and private** environment for women.
- Encourage **meaningful community interactions**
- Reduce harassment through **controlled access and moderation**

---

## Features (MVP)
- User authentication (sign up & login)
- Women-only access control
- Create posts
- Like and comment on posts
- User Profiles
- Report & block users
- Basic content moderation
